# Sales Dashboard in Excel (VBA Macro-Enabled)

## Overview
This project is an interactive sales dashboard built in Microsoft Excel using PivotTables, charts, and slicer-style navigation to analyze sales performance by city and sales executive.

## Key Features
- City-level navigation (e.g., Chennai, Delhi, Mumbai, Nagpur, etc.)
- Sales by Executive (Total Sales)
- Target Hit Percentage and Gap from Target
- Interactive visuals (bar chart, pie chart, trend line)

## Tools Used
- Microsoft Excel (PivotTables, Charts, Slicers/Buttons)
- VBA (.xlsm)

## How to Use
1. Download the `.xlsm` file
2. Open in Excel (Desktop)
3. Click **Enable Content / Enable Macros**
4. Use the city buttons to explore the dashboard
